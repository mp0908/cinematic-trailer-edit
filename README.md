# Cinematic Trailer Edit

🎬 **Project type:** Cinematic trailer-style edit  
🛠 **Tools:** Adobe Premiere Pro, After Effects  
📅 **Year:** 2024  

## Overview
This project is a cinematic trailer-style edit created by re-cutting scenes
from an existing feature-length film. The objective was to transform long-form
footage into a short, high-impact trailer format focused on pacing, tension,
and visual storytelling.

## My Role
- Full trailer edit from feature-length footage
- Scene selection & narrative pacing
- Rhythm-based cuts and transitions
- Sound design and audio timing
- Color correction and tonal consistency
- Minimal motion graphics

## Creative Direction
- Dark cinematic tone
- Controlled build-up and escalation
- Emphasis on atmosphere and impact moments
- Modern trailer editing structure
